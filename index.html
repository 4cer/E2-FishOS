What is FishOS?

Fish os is OS made completely in E2. In other word a version of lua used to make in game programs. Everything is made in gmod using wiremod. FishOS is os made for all types of gamemode. Most likely for darkrp.

Main goal

My main goal is to bring E2 programists together and make OS. I know that this is not easy but this is open source project. Everyone who will make working changes to code will be contributor

What you need to know to be contributor?

Don't make code that will be made only like that: if(ASDjh) { asdhkjas } else { asdhjkash }

You should use math mainly math e.g. EGP:egpBox(1,vec2(12,124),vec2(126,126)*ON)

Variables for making os: ON - value that is telling if pc is ON or OFF it can be only 1 or 0 Desk - value that is telling if PC has booted up or not it can only be 1 or 0
